# Fashion AI (fashion-ai)

Fashion AI is an AI-powered clothing in-painting service offered by Segmind. By accepting an input image and a natural-language prompt, the model edits the topwear or bottomwear region of the image and returns a modified image. It is useful for virtual styling, e-commerce mockups, and rapid wardrobe iteration.

## APIs

### Fashion AI

Segmind Fashion AI is an in-painting model that edits clothing in an image. It accepts an input image (auto center-cropped and resized to 512x512), a text prompt describing the desired edit, and an optional clothing region selector (topwear or bottomwear). Authentication is via the `x-api-key` header. The base URL is `https://api.segmind.com/v1` and the endpoint is `POST /fashion-ai`.
